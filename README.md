# 🚀 Hello World API – NestJS

This is a simple REST API built with [NestJS](https://nestjs.com/) using TypeScript.  
It returns a `Hello World!` message at the root endpoint (`/`).
## 📦 Tech Stack

- **Framework:** NestJS
- **Language:** TypeScript
- **Runtime:** Node.js
